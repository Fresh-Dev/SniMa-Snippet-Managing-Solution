1. Filename:
    - Name it like the shortcode of the target-language

2. Content:
    - Timestring specialities:
        - time_ago_string:  %1 = Numberic Value
                            %2 = Unit (e.g. Hour/Minute....)
                            Full example for "%1 %2 ago" => 2 hours ago
